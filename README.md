Après avoir simplement créer la structure du projet, nous avons commencé par mettre en place une règle sur GitHub : tout push doit d'abord être effectué sur une branche secondaire. Une Pull Request doit ensuite être validée avant que les changements soient intégrés dans la branche principale `main`.
